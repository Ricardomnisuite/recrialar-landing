Criar uma landing page premium para Google Ads com um único objetivo:

A página não é um website institucional. É uma página de conversão para pedidos de orçamento.

## Wizard de Orçamento

O wizard é o elemento mais importante da página.

Deve parecer uma conversa guiada premium, não um formulário longo.

Regras:

* Um passo de cada vez
* Barra de progresso visível
* Botão voltar
* Avanço automático nas escolhas simples
* Validação nos campos obrigatórios
* Design mobile-first
* Mensagem final enviada para WhatsApp com todos os dados preenchidos

---

## Passos do Wizard

### Passo 1 — Que serviço procura?

Opções:

* Remodelação completa
* Pintura interior
* Pintura em altura / rapel
* Pladur / tetos falsos
* Chão flutuante
* Barramento
* Pequenas reparações domésticas
* Manutenção de imóveis
* Outro

Se escolher **Outro**, mostrar campo obrigatório:

**Descreva o serviço que procura**

---

### Passo 2 — Onde fica o imóvel?

Opções:

* Setúbal
* Tróia
* Comporta
* Palmela
* Azeitão
* Sesimbra
* Outro

Se escolher **Outro**, mostrar campo obrigatório:

**Indique a localização**

### Passo 3 — Tipo de imóvel

Opções:

* Apartamento
* Moradia
* Condomínio
* Loja / Espaço comercial
* Escritório
* Outro

### Passo 4 — Dimensão aproximada

Opções:

* Pequena intervenção
* Até 30 m²
* 30–60 m²
* 60–100 m²
* Mais de 100 m²
* Não sei

### Passo 5 — Quando pretende iniciar?

Opções:

* Esta semana

* 1–3 meses
* Só quero orçamento

---

### Passo 6 — Tem fotografias do espaço?

Opções:

* Ainda não tenho

### Passo 7 — Descrição do trabalho

Campo textarea opcional.

### Passo 8 — Como nos encontrou?

Opções:

* Google
* Recomendação
* Facebook
* Outro

Se houver `gclid` ou `utm_source=google`, preencher automaticamente como Google Ads e saltar este passo.

Campo obrigatório.

## Performance

Obrigatório:

* Lighthouse target acima de 95
* LCP abaixo de 2.5s
* CLS abaixo de 0.1
* Imagens otimizadas
* Lazy loading
* Google Maps com facade/lazy-load
* Sem scripts pesados desnecessários
* Sem 3D pesado
